## 💡 使用说明

### 如何添加 milestone 到关卡

以 P8 为例：

```javascript
// 在 p8-multi-agent.js 的 easy.sections 数组末尾添加：

easy: {
  sections: [
    { type: 'story', ... },
    { type: 'concept', ... },
    // ... 其他 sections
    { type: 'quiz', ... },
    
    // 添加里程碑（从 p8-milestone.js 复制）
    {
      type: 'milestone',
      icon: '🎓',
      title: '里程碑 1：Agent 基础大师',
      // ... 其他内容
    }
  ]
}
```

对 P16、P21 做同样的操作。
