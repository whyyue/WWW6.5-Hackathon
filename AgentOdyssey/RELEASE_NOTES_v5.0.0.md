- 🚀 深度探索者：完成所有深度篇（P9-P16）

- 🐟 群体智能专家：完成 MiroFish 篇（P23-P25）

**功能特性**：

- 地图页面展示
- 锁定/解锁状态视觉区分

---

### 2. 🎓 里程碑系统

**新增 4 个阶段性里程碑**：

#### 里程碑 1：Agent 基础大师（P8 完成后）
- 成就：掌握 LLM、工具、ReAct、记忆、Multi-Agent
- 项目：天气查询 Agent
- 下一步：深度篇或实战篇

#### 里程碑 2：生产级 Agent 开发者（P16 完成后）
- 成就：掌握 Transformer、预训练、后训练、框架
- 项目：智能代码审查系统
- 下一步：帝国篇或 MiroFish 篇

#### 里程碑 3：Multi-Agent 系统架构师（P21 完成后）
- 成就：掌握三省六部、制度化协作
- 项目：智能客服系统（3-Agent）
- 下一步：MiroFish 篇

#### 里程碑 4：全栈 Agent 开发者（P27 完成后）
- 成就：掌握 GraphRAG、人格系统、仿真引擎
- 项目：mini MiroFish
- 下一步：持续学习

**每个里程碑包含**：
- 成就展示
- 技能清单
- 里程碑项目（详细需求 + 提示）
- 下一步建议

---

### 3. 🎮 互动元素（P23-P25）

**新增 3 种互动类型**：

#### Challenge（代码挑战）
- P23：实现文本分块函数
- P24：设计 Agent 人设
- P25：设计活动配置

#### Debug（调试任务）
- P23：修复图谱构建代码
- P25：修复并发问题

#### Sandbox（沙盒实验）
- P23：调整分块参数
- P24：调整人格参数
- P25：调整仿真参数

**功能特性**：
- 代码编辑器（textarea）
- 参数滑块（range input）
- 实时可视化
- 提示系统

---

### 4. 📚 MiroFish 系列（P23-P25）

**新增 3 个关卡**：

#### P23：知识图谱星 🕸️
- 简单模式：GraphRAG vs 传统 RAG
- 困难模式：本体设计、文本分块、实体提取
- 地狱模式：微软 GraphRAG 论文（2024）
- 互动：Challenge + Debug + Sandbox

#### P24：Agent 人格星 🎭
- 简单模式：人格的四个维度
- 困难模式：OASIS Profile、人设生成流程
- 地狱模式：Generative Agents 论文（Stanford 2023）
- 互动：Challenge + Sandbox

#### P25：仿真引擎星 ⚙️
- 简单模式：调度器、规则引擎、状态管理
- 困难模式：活动配置、事件循环、IPC 通信
- 地狱模式：OASIS 论文（CAMEL-AI 2024）
- 互动：Challenge + Debug + Sandbox
