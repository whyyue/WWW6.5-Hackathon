# SheProves · 她证

一个面向高中女生的数学互助与知识变现平台。

SheProves 希望打破"女生不擅长数学"的观念。我们不只是想帮女生学懂数学，更想让女生在理解、表达和互助中，成为数学知识的翻译者和传递者——并且靠这个能力赚到钱。

---

## 一句话介绍

高中女生把教科书的数学"翻译"成人话，同伴投票"我听懂了"，讲得好的人赚 AVAX——在彼此讲解中打破"女生不擅长理科"的刻板印象。

---

## 我们想解决什么问题

很多女生并不是学不会数学，而是在传统数学教育中，缺少更适合自己的进入方式。

- 数学概念抽象，课堂往往只有一种讲法
- 第一次没听懂就容易归因于"我不擅长数学"
- 补课 ¥200/小时，讲法和课堂重复
- 很少有空间鼓励学生用自己的语言重新解释数学
- 会讲数学的女生没有任何渠道用这个能力赚钱
- 女生在理科中的能力经常被低估，缺少可验证的数据来反驳

我们相信：

**不是女生不擅长数学，而是数学可以被用更多方式重新解释。**

---

## 核心想法

SheProves 建立在四个核心点上：

### 1. 多角度理解——翻译，不是做题

同一个数学知识点，不应该只有一种标准讲法。SheProves 的核心动作是"翻译"——把教科书语言翻译成自己的话：

```
┌─────────────────────────────────────────┐
│ 📖 教材原文                              │
│ "函数 f(x) 在 x₀ 处可导，当且仅当       │
│  极限 lim[f(x₀+Δx)-f(x₀)]/Δx 存在"     │
│                                           │
│ ────── ✨ 她的翻译 ──────                 │
│                                           │
│ "你在山坡上骑车，导数就是脚下这一小步      │
│  有多陡。Δx 是你迈的步子，越小越准。"     │
│                                           │
│ ✅ 312 人听懂 · 💰 已赚 0.8 AVAX          │
└─────────────────────────────────────────┘
```

同一个知识点可以有 50 种翻译——类比型、图像型、生活化型。目标不是灌输统一答案，而是帮她找到"终于听懂的那个角度"。

### 2. 同伴讲解比老师更容易听懂

很多时候，真正让人突然听懂的，不是标准解析，而是一个处在相近学习阶段的人，用更贴近日常语言讲出来的解释。

### 3. 讲解本身就是学习

当一个女生把数学讲给另一个女生听，她也在完成一次费曼学习法练习——整理概念、简化表达、验证理解。讲解不是学习之外的附加动作，而是学习本身的一部分。

### 4. 讲数学能赚钱

会讲数学的女生不应该只是"帮忙"，她的能力值得被定价。通过悬赏和打赏机制，讲得好的人赚到真金白银。

## 产品机制

### 翻译 Feed

首页是一个 Feed，每张卡片展示教材原文和她的翻译对照：

- 按知识点筛选（函数 / 数列 / 概率 / 导数）
- 按"听懂了"票数排序
- 看到好的翻译可以打赏

### 悬赏广场

不懂的人贴一段教材原文 + 附 AVAX 悬赏，多位女生提交翻译，社区投票"听懂了"，求助者选择最佳翻译。合约自动分账：90% 给翻译者，10% 给平台。48 小时超时则得票最高者自动获奖。

### 讲给下一位女生听

每次看完翻译后，可以留下自己的版本——用自己的话解释这个知识点，说出自己是怎么想通的。下一位用户进入时，就能看到前面女生留下的讲解。

这形成一个"女生帮助女生理解数学"的接力链。

### 个人档案

- 我的翻译墙（发布过的所有翻译）
- 能力地图（擅长哪些知识点、验证通过率）
- SBT 成就列表
- 收入记录
- $PROVE 声誉分

### 全局数据面板

读取链上 STEMIndex 合约，实时展示：

- 多少位女生在贡献翻译
- 多少条翻译被验证为"有效教学"
- 累计发放多少悬赏

这个数据任何人可从链上读取——不是平台说"女生数学很棒"，是链上数据在说。

## 经济模型

### 女生怎么赚钱

- **悬赏**：回答悬赏被选中 → 拿走 90% 赏金
- **打赏**：好的翻译收到打赏 → 拿走 95%
- **品牌签约**：声誉高的翻译者被教培品牌招募为兼职讲师

### 平台怎么赚钱

- 悬赏抽成 10%
- 打赏抽成 5%
- 教培品牌合作费（赞助悬赏 + 人才招募）
- 升学能力档案增值服务

### $PROVE 声誉代币

$PROVE 不是钱，是教学信用（不可交易）。翻译被验证 → 赚 $PROVE。声誉高者排名靠前、被品牌优先签约、链上可查用于升学申请。

## 为什么上链

1. **悬赏资金锁在合约里**，不经过平台账户，选中后自动分账
2. **"312 人听懂了"是链上共识**，不是数据库数字，任何人可验证
3. **收入记录透明**，每笔悬赏和打赏的流向链上公开可查
4. **STEMIndex 合约实时公开**——"女生能教数学"变成链上可查的事实

---

## 为什么是 Avalanche

每次"听懂了"投票、每笔打赏都是独立的链上交易。活跃社区意味着每天数千次写入。

- Gas < $0.01/tx → 小额打赏经济可行
- 亚秒确认 → 投票和打赏即时到账

**Subnet 愿景：教学即挖矿。** Avalanche Subnet 允许自定义验证者和 gas 代币。SheProves Education Subnet 的设计：

- 验证者不是质押资金的人，而是教学声誉最高的女生
- Gas 用 $PROVE 支付，通过教学赚取，不用购买
- 在这条链上，**教数学的能力就是算力**

当前 MVP 在 C-Chain 上用合约模拟这套机制，未来迁移到独立 Subnet。

SheProves 不是一个普通的数学工具。
