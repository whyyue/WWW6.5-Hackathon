# 🌱 项目名：Symbreath 共生呼吸

---

## 📝 简介（痛点与背景 / Overview）

<details>
<summary>中文 / 点击展开</summary>

我们利用 Web3 的博弈机制与实时气象数据，建立一套跨物种的共生协议。  
通过将森林的生存压力转化为个体的冥想交互，让环保从“遥远的公益”变成“切身的感知”。  
用户可以通过链上抵押代币完成冥想挑战，如果挑战成功将退回代币并铸成 NFT 徽章，中途断签则质押金自动捐给真实环保慈善地址。

</details>

<details>
<summary>English / Click to expand</summary>

We leverage Web3 game-theory mechanics and real-time weather data to create a cross-species symbiosis protocol.  
By converting forest survival stress into individual meditation interactions, environmental protection becomes a tangible, personal experience.  
Users stake tokens on-chain to complete meditation challenges. Successful completion refunds the tokens and mints an NFT badge, while failure or missed check-ins automatically donate the stake to real-world environmental charities.

</details>

## ⚡ 功能 / Features

<details>
<summary>中文 / 点击展开</summary>

- 💰 **质押挑战:** 抵押 0.01 ETH，定义 7 / 21 天挑战周期  
- 🧘 **每日打卡:** 选择灾难模式触发冥想动效  
- 🏆 **挑战成功:** 全额退款 + 铸造专属 SBT 徽章  
- ❌ **主动放弃:** 质押金自动捐给环保机构  
- 🔗 **SBT 灵魂绑定:** 不可转让  

</details>

<details>
<summary>English / Click to expand</summary>

- 💰 **Stake Challenge:** Stake 0.01 ETH for 7 / 21-day challenges  
- 🧘 **Daily Check-in:** Choose disaster mode to trigger meditation animations  
- 🏆 **Success Reward:** Full refund + exclusive SBT badge  
- ❌ **Voluntary Exit:** Stake automatically donated to charity  
- 🔗 **Soulbound Token (SBT):** Non-transferable  

</details>

## 📊 Demo 流程 / Demo Flow

<details>
<summary>展开 / Click to expand</summary>

1. 用户选择灾难模式（干旱 🌞 / 洪水 🌊 / 热浪 🔥）
